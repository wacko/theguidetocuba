Antes de Empezar

Este libro es para cualquier programador que quiera comenzar a desarrollar una aplicación Cuba desde cero, no importa si eres experimentado o no. A través de los capítulos, construiremos un clon de Twitter.

Para probar los ejemplos del libro, necesitas previamente tener instalado:

  • Ruby 2.0 o superior. La última versión la puedes encontrar en la página de descargas de Ruby.

  • La base de datos Redis. Si aún no has instalado Redis, puedes seguir las instrucciones en la página de descargas de Redis.

  • Git, el software de control de versiones. Puedes seguir las instrucciones de instalación que se encuentran en la sección de introdución de Pro Git.

NOTA: Todos los ejemplos de código han sido probados tanto en OSX como en Linux.
